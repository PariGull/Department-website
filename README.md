# Department of Computer Science Website

## Overview
This is a **multi-page departmental website** built using **HTML 4.01 Transitional** only.  
.  

The website is fully functional in modern browsers and includes optional pages for a complete experience.



## Folder Structure

DepartmentWebsite/
│
├── index.html        → Home Page
├── about.html        → About Department
├── faculty.html      → Faculty Members
├── labs.html         → Laboratory Information
├── courses.html      → Courses Offered
├── gallery.html      → Department Gallery
├── contact.html      → Contact Form
│
└── images/           → Folder containing all icons
    ├── home.png
    ├── about.png
    ├── faculty.png
    ├── labs.png
    ├── courses.png

## Features

### 1. Navigation Bar
- Present on every page.
- Includes **icons** for each section (Home, About, Faculty, Labs, Courses, Gallery, Contact).
- Table-based layout for professional look.

### 2. Home Page (`index.html`)
- Welcome message with department overview.
- Clean layout with professional header and footer.

### 3. About Page (`about.html`)
- Department introduction with **vision & mission** lists.
- Chairperson message in **blockquote**.
- Use of headings and text formatting.

### 4. Faculty Page (`faculty.html`)
- Faculty table including:
  - Name
  - Designation
  - Qualification
  - Email
  - Office hours
- Table features: **rowspan** and **colspan**
- Alternating row colors for professional look.

### 5. Labs Page (`labs.html`)
- Lists of lab names and equipment.
- Includes **two lab images with captions**.
- Use of `<ul>` and `<ol>` for lists.

### 6. Courses Page (`courses.html`)
- Table of courses with code, title, and credit hours.

### 7. Gallery Page (`gallery.html`)
- Displays department events and labs in **aligned image grid**.
- Captions included for each image.

### 8. Contact Page (`contact.html`)
- Form includes:
  - Name, Email, Phone (text inputs)
  - Program (dropdown/select)
  - Gender (radio buttons)
  - Interests (checkboxes)
  - Message (textarea)
  - Submit & Reset buttons

---

## Technologies Used
- **HTML 4.01 Transitional** (no HTML5 or CSS used)
- Table-based layout for headers, navigation, content boxes, and footer
- Web-based icons and images
- Browser-compatible in Chrome, Edge, Firefox

---

## Instructions to Run

1. Download or unzip the `DepartmentWebsite/` folder.
2. Open `index.html` in any web browser.
3. Navigate through all pages using the **navigation bar**.
